# Monthly Menu Update Guide

**Quick reference for updating menus each month**

## 🗓️ Before Each Pop-Up

### 1. Prepare Your Images

- [ ] Create/export dessert menu image (any format: JPG, PNG, etc.)
- [ ] Create/export drink menu image (any format: JPG, PNG, etc.)
- [ ] Images can be any size (will be optimized automatically)
- [ ] Save images to your computer with recognizable names

### 2. Update the Website

#### Step 1: Go to Admin
**URL**: https://myzeste.com/admin

#### Step 2: Sign In
Click "Sign in with Google" and use: **admin@myzeste.com**

#### Step 3: Upload Dessert Menu
1. Under "Upload New Menu" → "Dessert Menu"
2. Click "Choose image file"
3. Select your dessert menu image
4. Preview will appear
5. Click "Upload Dessert Menu"
6. Wait for "Menu uploaded successfully!" message
7. Page will refresh automatically

#### Step 4: Upload Drink Menu
1. Under "Upload New Menu" → "Drink Menu"
2. Click "Choose image file"
3. Select your drink menu image
4. Preview will appear
5. Click "Upload Drink Menu"
6. Wait for "Menu uploaded successfully!" message
7. Page will refresh automatically

#### Step 5: Verify
1. Check the admin dashboard shows your new images
2. Visit https://myzeste.com/menu to confirm
3. Test on mobile device
4. Done! ✨

## ⏱️ Time Required

**Total time**: ~2-3 minutes
- Signing in: 30 seconds
- Uploading dessert menu: 30 seconds
- Uploading drink menu: 30 seconds
- Verification: 30 seconds

## 🆘 Quick Troubleshooting

### Can't sign in?
- Make sure you're using admin@myzeste.com
- Try incognito/private browsing mode
- Clear browser cache

### Upload button disabled?
- Make sure you selected an image file
- Check file is an image (JPG, PNG, etc.)

### Image not showing after upload?
- Wait for success message
- Hard refresh page (Cmd+Shift+R or Ctrl+Shift+R)
- Check https://myzeste.com/menu in new tab

## 💡 Pro Tips

1. **Prepare images in advance** - Have them ready the day before
2. **Use consistent dimensions** - Makes menu look professional
3. **Test on mobile** - Most customers view on phones
4. **Upload during off-hours** - Avoid updating during peak traffic
5. **Keep originals** - Save original files as backup
6. **Name files clearly** - e.g., "February_2026_Dessert.jpg"

## 🎨 Image Best Practices

### Recommended Specs:
- **Format**: JPG or PNG
- **Width**: 1080-2000 pixels
- **Aspect ratio**: 3:4 (portrait) or 2:3
- **File size**: Under 5MB (smaller = faster loading)
- **Quality**: High quality, readable text

### Design Tips:
- ✅ High contrast for readability
- ✅ Large font sizes for mobile
- ✅ Clear item names and prices
- ✅ Professional photos
- ✅ Consistent branding
